# ⚡ WiFi Voucher Hunter V2

A professional-looking WiFi Voucher Brute Force tool for educational purposes and security testing. This script uses Python to automate the process of guessing 6-digit WiFi vouchers.

## 🚀 Features
* **Hacker UI:** Cool ASCII Art and colored interface for Termux.
* **Real-time Progress:** Live scanning animation and percentage display.
* **Auto-Save:** Found vouchers are automatically saved to `found_vouchers.txt`.
* **Efficient:** Lightweight and fast processing.

## 📸 Interface Design
* Hacker Face Banner (Red/Cyan)
* Dynamic Spinner Animation
* Progress Tracking [ 00.00% ]
