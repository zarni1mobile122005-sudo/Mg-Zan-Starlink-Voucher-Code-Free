> **The Ultimate Automated Voucher Bruteforcer for Termux.**

`WiFi Voucher Hunter V2` သည် Termux အသုံးပြုသူများအတွက် အထူးထုတ်လုပ်ထားသော WiFi Login Voucher guessing tool ဖြစ်သည်။ လှပသော Hacker UI Design နှင့်အတူ အားကောင်းသော Brute-force အင်ဂျင်ကို ပေါင်းစပ်ပေးထားသည်။

---

## ✨ စိတ်လှုပ်ရှားဖွယ် Features များ
* 🖋️ **Dynamic Branding:** Startup မှာ "Mg Zan" စာသားကို ရေးလိုက်၊ ဖျက်လိုက် လုပ်ပေးမည့် Typing Animation။
* 💀 **Hacker Dashboard:** အဆင့်မြင့် ASCII Art နှင့် Neon Color Themes များ။
* 📊 **Live Analytics:** Scanning ပြုလုပ်နေစဉ် Progress Percentage (%) နှင့် Dot-Spinner Animation။
* 💾 **Smart Auto-Log:** Valid Voucher တွေ့ရှိပါက `found_vouchers.txt` ထဲသို့ အလိုအလျောက် သိမ်းဆည်းပေးခြင်း။
* ⚡ **Optimized Speed:** Connection timeout များကို ထိန်းညှိပေးထားသဖြင့် စက်မလေးဘဲ မြန်မြန်ဆန်ဆန် အလုပ်လုပ်နိုင်ခြင်း။
